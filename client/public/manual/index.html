    Transfera — Technical Architecture Manual

    Systems guide to the P2P file sharer: Next.js frontend on GitHub Pages, C++ HTTP API on Render, coordinated P2P transfers, configurable download limits, and upload validation.

    New here? Start with Product features for current behavior, then System architecture for design. On mobile, tap ☰ Menu to open navigation.

    What Transfera does

    A user uploads a file (max 5 MB, no videos), sets max downloads (default 1), and receives an invite code (random high port number). Recipients enter the code; each successful download runs an internal P2P transfer and counts toward the limit. When the limit is reached, the invite and temp file are removed.
