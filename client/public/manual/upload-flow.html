    Upload pipeline

    POST /api/upload — multipart file + optional maxDownloads → invite JSON.

    1. Sequence

    Browser httplib worker FileController Disk FileSharer │ │ │ │ │ │ POST multipart │ │ │ │ │ file + maxDownloads │ │ │ │ ├───────────────────────►│ parse form │ │ │ │ ├───────────────────►│ handleUpload │ │ │ │ ├─────────────────►│ write file │ │ │ ├──────────────────────────────────►│ offerFile(N) │ │◄───────────────────┤ 200 {port,max} │ │ │◄───────────────────────┤ │ │ │ │ (no P2P listener │ │ │ │ │ started on upload) │ │ │ │

    2. Multipart fields

    FieldRequiredNotes
    fileYesBinary content; non-empty
    maxDownloadsNoInteger 1–100; default 1 if missing
    server/src/services/fileController.cpp — parseMaxDownloadsField

    3. Server validation

    1. POST only; multipart/form-data
    2. Field file present and non-empty
    3. Client should already enforce 5 MB and no video (not re-checked on server today)
